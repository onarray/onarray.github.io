W3C CSS Validator guide

https://github.com/tlvince/w3c-css-validator-guide

The W3C CSS Validator guide provides a simple walk-through to build the W3C CSS validation service from its source.

Commisioned by the developers of SiteValidator for the second time, onarray‘s technical writing proved to be simple and accessible to a wide audience.

We used a Vagrant managed virtualised environment in order to guarantee reproducibility and handed the guide back to the community via an open source license.
